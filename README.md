# 🛡️ CipherFile v2.0 - Asenkron AES Güvenliği

Bu proje, dosyaları **AES-256** algoritması ile şifrelemek ve orijinal dosyayı güvenli bir şekilde imha etmek için geliştirilmiş bir güvenlik aracıdır.

## 🚀 Öne Çıkan Özellikler
* **Asenkron İşlem:** Şifreleme sırasında donma yapmaz (Threading kullanıldı).
* **Güvenli Silme (Wiping):** Orijinal dosyanın üzerine rastgele veriler yazılarak silinir.
* **Modern Arayüz:** CustomTkinter ile karanlık mod desteği.

## 🛠️ Kurulum
1. Python yüklü olduğundan emin olun.
2. Gerekli kütüphaneleri yükleyin:
   `pip install pyAesCrypt customtkinter`
